# LearnTrack - Student & Course Management System

## Project Description
LearnTrack is a console-based Student & Course Management System built using Core Java. It allows administrators to manage students, courses, and enrollments in an educational institution.

## Features
- **Student Management**: Add, view, search, and deactivate students
- **Course Management**: Add, view, activate/deactivate courses
- **Enrollment Management**: Enroll students in courses, track enrollment status
- **Data Persistence**: In-memory storage using ArrayLists
- **Error Handling**: Custom exceptions and input validation

## How to Compile and Run

### Prerequisites
- Java JDK 11 or higher

**Set Environment Variables**
   - Right-click "This PC" → Properties → Advanced System Settings
   - Click "Environment Variables"
   - Under System Variables, click "New"
     - Variable name: `JAVA_HOME`
     - Variable value: `C:\Program Files\Java\jdk-11.0.xx`
   - Edit "Path" variable, add: `%JAVA_HOME%\bin`

 **Verify Installation**
   ```cmd
   java -version
   javac -version
